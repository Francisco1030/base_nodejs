<h2 align="center"> TDD </h2>

#### Dependências

* NodeJs
* NPM
* MOCHA
* NOCK

------------

#### Comandos

###### Baixar as dependecias

``` shell
    npm install
```

###### Rodar o teste

``` shell
    mocha tests.js
```

#### Código

###### Mocha

Em `01/tests.js` , temos um exemplo de teste usando o `mocha` :

``` js
    describe('Decrição do teste', function () {
        ...
    })
```

Para criar função de teste:

``` js
    it('deve fazer algo', async () => {
        ...     
    })
```
